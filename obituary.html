Working with a partner, we created an obituary application that allows the user to create obituaries for the deceased. The application has an AWS backend, which was configured with Terraform. The ChatGPT api was implemented to write the obituaries based on user inputs.

Specs

  • Software: React, Python, Terraform
  • Dependencies: None
  • Code Repository:  

Project Details

This project was completed as the final assignment for ENSF 381 at the University of Calgary.

This obituary application was created in React, with a backend on AWS. The AWS backend was configured through Terraform. Many AWS services were used in the creation of the web application, including:

  • Lambda
  • Cloudinary
  • AWS Systems Manager Parameter Store
  • Amazon Polly
  • Dynamo DB

AWS services such as AWS Systems Manager Parameter Store were used for security to store private keys used to access the AWS functions and database as well as the ChatGPT api.

When creating a new obituary, the user is prompted to enter information about the deceased. This information is passed through a chatGPT prompt where an obituary is automatically created. Using amazon polly, the user may listen to the obituary as it is read to them. Users may open and close different obituaries as they wish. Obituaries are stored on the AWS backend.
